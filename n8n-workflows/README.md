# n8n Workflows

This project uses n8n for backend automation. Import these workflows:

## Workflows Needed:
1. **Get Security Tasks** - GET /security-tasks
2. **Update Task Status** - POST /update-task  
3. **Get Progress Stats** - GET /progress
4. **CSV Export** - GET /export

## Setup:
1. Import each workflow JSON into n8n
2. Configure PostgreSQL connection
3. Activate workflows
